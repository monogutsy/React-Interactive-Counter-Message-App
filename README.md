# 🎯 React Interactive Counter Message App

<p align="center">
  A simple React application that allows users to increase, decrease, and reset a counter while displaying live input messages and conditional goal notifications.
</p>

<p align="center">
  ⚛️ React • 💬 Live Input
</p>

---

<p align="center">
  <a href="https://react-interactive-counter-message-a-psi.vercel.app/">
    🔗 Live Demo
  </a>
</p>

---

# ✨ Features

✅ Increase counter dynamically  
✅ Decrease counter dynamically  
✅ Reset counter instantly  
✅ Live message input display  
✅ Conditional rendering  
✅ Displays “Goal Reached!” when count ≥ 5

# 🚀 Getting Started

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/Stylish-Counter-App.git
```

---

## 2️⃣ Navigate to the Project Folder

```bash
cd Stylish-Counter-App
```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

---

## 4️⃣ Start the Development Server

```bash
npm start
```
